# _::JAVIER DIEZ BRIZ::_ FizzBuzz

## Requirements

* docker
* docker compose
* make
* git

## Setup

Clone repository

```
git clone git@github.com:jv10/fizzbuzz.git
```

Install app

```
make install
```

Show aditional options

```
make help
```

## ROUTES

* [GET] http://127.0.0.1:8089/desafio/fizz/buzz

## TEST

Run Unit, Integration and E2E test Suits.

```
make run-test
```
